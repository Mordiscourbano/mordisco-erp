# Sprint 8 — Cocina y KDS

Incluye:

- Pantalla de cocina.
- Estados: pendiente, preparando, listo y entregado.
- Actualización en tiempo real.
- Alertas visuales por demora.
- Métricas de preparación y entrega.
- Historial de cambios de estado.

## Instalación

1. Crear rama `sprint-8-kds`.
2. Ejecutar `supabase/sprint-8-kds.sql`.
3. Subir `src`, `supabase` y `README.md`.
4. Commit: `Sprint 8: Cocina y KDS`.
5. Probar `/cocina` y `/cocina/metricas`.

No hay que copiar CSS manualmente.
