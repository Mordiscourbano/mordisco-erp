# Mordisco ERP — Sprint 6: Compras y Proveedores

Incluye proveedores, órdenes de compra, recepción total/parcial, ingreso automático al stock, actualización opcional de costos, historial de precios y compra sugerida.

## Instalación

1. Crear rama `sprint-6-compras` desde `main`.
2. Ejecutar `supabase/sprint-6-compras.sql` en Supabase.
3. Subir `src`, `supabase` y `README.md` a la rama.
4. Copiar `src/app/sprint-6-additions.css` al final de `src/app/globals.css` y borrar el temporal.
5. Probar `/proveedores`, `/compras`, `/compras/nueva`, `/compras/recepcion` y `/precios-historicos`.
